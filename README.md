# Friends on Fire!

A side-scrolling space shooter built with Pygame. Survive as long as you can, destroy asteroids, and collect weapon upgrades.

## Quick Start

Requires **Python 3.10+**.

```bash
git clone <repo-url>
cd friends_on_fire
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python Game.py
```

## Controls

| Key | Action |
|-----|--------|
| W/A/S/D or Arrow keys | Move |
| Space | Shoot |
| P | Fire secondary weapon |
| Enter | Confirm / Pause |
| Escape | Pause (in game) |
| Up/Down | Navigate menus |
